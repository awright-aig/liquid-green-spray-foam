07 — Social

Social Media Icons

All social media icons (Facebook, Instagram, Yelp) use the same visual treatment: their original colored source images are transformed to lime-green via a CSS filter, creating a unified monochromatic appearance. On hover, they invert to near-black against a lime-green background.

Default State (Lime-Green)

Hover State (Inverted on Lime)

CSS Filter

/* Default: transforms any colored icon to lime-green */
filter: brightness(0) saturate(100%) invert(72%) sepia(50%) saturate(500%) hue-rotate(40deg) brightness(95%) contrast(90%);

/* Hover: transforms to near-black */
filter: brightness(0) saturate(0%) invert(10%);

Placement

  • Sidebar: Fixed left edge, vertically centered, 40×40px per icon (desktop only, hidden on mobile)
  • Footer: Inline row within the first column, 36×36px per icon with border

Every interactive element across the site shares a single consistent hover effect defined by three CSS custom properties. This includes cards, buttons, nav links, social icons, FAQ items, gallery items, and carousel arrows.

10 — Rules

Usage Rules

Logo Usage

Do

  • Use the full logo (logo-transparent.png) for hero sections and large placements
  • Use the spray gun icon (menu-logo.png) for navigation headers at 46px height
  • Use the spray gun favicon for browser tabs and bookmarks
  • Maintain transparent backgrounds on all logo variants
  • Apply drop-shadow on hero logos over video/image backgrounds
  • Keep adequate clear space around all logo marks

Don't

  • Don't stretch, skew, or distort any logo variant
  • Don't change the logo colors from the original lime-green palette
  • Don't place logos on light backgrounds without a dark container
  • Don't use the full logo at small sizes where text becomes illegible
  • Don't add rounded corners, gradients, or effects not in this guide
  • Don't recreate or substitute the logos with different typefaces

Color Usage

Do

  • Use Lime (#ACD155) as the primary accent for all headlines, borders, and CTAs
  • Use Charcoal (#262626) as the dominant background color
  • Use Purple (#8138B7) sparingly for secondary highlights and franchise content
  • Use rgba transparency values for subtle borders and hover states
  • Maintain high contrast between lime elements and dark backgrounds

Don't

  • Don't introduce new colors outside the defined palette
  • Don't use lime as a background for large text areas (readability issues)
  • Don't use white as a background — the brand is dark-first
  • Don't apply purple to primary navigation or main headlines
  • Don't use saturated lime on lime (no lime text on lime background)

Typography Usage

Do

  • Use Rubik Distressed exclusively for h1 and h2 headlines
  • Use Oswald for h3 sub-headings within content sections
  • Use Raleway for navigation, eyebrow labels, and button text
  • Use Poppins for all body copy and paragraph text
  • Maintain uppercase on all headlines, navigation, and buttons

Don't

  • Don't use Rubik Distressed for body text or small sizes
  • Don't mix font families in the same hierarchy level
  • Don't use lowercase for headlines or navigation items
  • Don't substitute system fonts for the specified Google Fonts
  • Don't reduce letter-spacing below the defined values

Iconography Rules

Do

  • Use the standard SVG style: stroke #ACD155, width 1.5, square caps
  • Display icons in a bordered box with lime border
  • Keep icon sizes consistent: 26px within 50px icon boxes
  • Apply the same CSS filter to social icons for lime-green uniformity

Don't

  • Don't change icon stroke color from #ACD155
  • Don't fill icons (they are stroke-only, no fill)
  • Don't use rounded line caps (must be square)
  • Don't show social icons in their original brand colors
